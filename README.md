Audio → Text → Voice

An end-to-end pipeline that converts speech audio into text and then generates voice output from the transcribed text.
This project demonstrates speech recognition (ASR) and text-to-speech (TTS) working together in a single workflow.

🚀 Features

🎧 Audio Input – Upload or record speech audio

📝 Speech-to-Text (ASR) – Converts audio into accurate text transcription

🔊 Text-to-Speech (TTS) – Generates natural-sounding speech from text

⚡ Fast Processing – Lightweight pipeline for quick conversion

🧩 Modular Architecture – Easy to swap ASR or TTS models
